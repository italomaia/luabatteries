Lua Batteries
=============

Do you like map, reduce, filter and other cool builtin functions from python? Would you like to use them in lua? Well,
lua batteries do just that for you.

## Supported functions

* map(fnc, table)
* filter(fnc, table)
* all(table)
* any(table)
* len(table)
* reduce(fnc, table)
* range(size)
* slice(table, start, stop)
* sum(table)
* cmp(a, b)
